# Style

Use this file when drafting, rewriting, or polishing prose.

## Language

- Write in Chinese by default.
- Use clear, precise, and natural Chinese.
- Avoid empty slogans, inflated wording, and generic AI-like transitions.
- Prefer concrete explanation over abstract claims.

## Tone

- Be rigorous but readable.
- Sound like a patient engineer or analyst explaining a hard topic clearly.
- Avoid sensationalism, overconfidence, and unnecessary drama.
- Keep claims modest when evidence is limited.

## Explanation Style

- Start from the problem, then explain the concept, structure, mechanism, and implication.
- Introduce jargon only when needed.
- Do not use unexplained terminology to explain new terminology.
- Use examples, analogies, diagrams, and tables to reduce cognitive load.
- Prefer 图文并茂 when it improves understanding: break up dense explanations with useful diagrams, tables, timelines, comparison visuals, or image prompts instead of producing a dry wall of text.

## Formatting Style

- Use paragraphs for reasoning.
- Use lists for steps, comparisons, checklists, or dense factual points.
- Avoid excessive bold text, blockquotes, nested lists, decorative formatting, and emoji.
- Keep section titles specific and useful.
